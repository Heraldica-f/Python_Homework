# Проект автотестов на Python при использовании Selenium + Allure

## Описание проекта

Проект содержит автоматизированные тесты для веб-приложений с использованием:
- **Python** + **pytest** - для написания и запуска тестов
- **Selenium** - для автоматизации браузера
- **Allure** - для создания наглядных отчетов
- **Page Object Pattern** - для структурирования кода

### Тестовые сценарии

1. **Тестирование "медленного" калькулятора** (`test_calc.py`)
   - Открытие калькулятора
   - Настройка задержки вычислений
   - Выполнение операции "7 + 8 ="
   - Ожидание и проверка результата (15)

2. **Тестирование интернет-магазина** (`test_shop.py`)
   - Авторизация пользователя
   - Добавление товаров в корзину
   - Оформление заказа
   - Проверка итоговой суммы

## Быстрый старт

1. **Базовый запуск без формирования отчета**
- # Запустить все тесты:
        pytest
- # Запустить конкретный тест:
        pytest test_calc.py
        pytest test_shop.py
- # Запустить с подробным выводом
        pytest -v

2. **Запуск тестов с формированием отчета в allure**
- # Шаг 1: Запустить тесты и собрать данные для отчета:
        python -m pytest tests/ --alluredir=allure-results
- # Шаг 2: Сгенерировать и открыть отчет
        allure serve allure-results

### Предварительные требования

- Python>=3.14+
- Установленный браузер Chrome и/или Firefox, а также вебдрайверы для каждого браузера
- Pytest>=9.0.2
- Установленные библиотеки Selenium>=4.40 и Allure-pytest>=2.15+
